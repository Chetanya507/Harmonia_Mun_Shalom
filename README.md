# UCSF – Union of Culture & Sports Fest Website
This is a website for the **Union of Culture & Sports Fest (UCSF)**, an inter-school festival organized by **Shalom Hills International School**. The site showcases festival details, student houses (dynasties), event highlights, and schedules, providing visitors with an immersive experience of the fest.
By *Ean Kotadia – Class 11 B* | UCSF Project

---

## Features

- **Hero Section:** Engaging introduction to the festival.
- **About Section:** Explains the purpose and values of UCSF.
- **Houses/Dynasties:** Displays the four student houses with descriptions and colors.
- **Fest Highlights:** Day-wise events, competitions, and award ceremonies.
- **Responsive Design:** Optimized for desktops, tablets, and mobile devices.

---

## Technologies Used

- **React.js** – Front-end library for building UI components.
- **CSS** – Styling and layout of pages and cards.
- **JavaScript** – Interactive elements and functionality.

## Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/yourusername/ucsf-website.git
```
2. Navigate to the project directory:
```cd ucsf-website```

3. Install dependencies:
```npm install```

4. Start the development server:
```npm start```
